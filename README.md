# C.A.S.T. – Chœur Artistique & Spirituel de Tanà

✨ Ce site vitrine présente l’univers du groupe évangélique C.A.S.T., fondé en 2003 à Antananarivo.

🎶 Son essence : une harmonie entre musique sacrée, prière vivante et excellence vocale.

📐 Conception modulaire, responsive et esthétique, réalisée avec [TailwindCSS](https://tailwindcss.com) et JavaScript.

👤 Création par : [Tovoniaina RAHENDRISON](mailto:positifaid@live.fr)

🔗 Accès en ligne : [https://LeMizoo.github.io/cast-site/](https://LeMizoo.github.io/cast-site/)

## 📂 Arborescence
- `index.html` – Page principale du site
- `css/style.css` – Feuille de style
- `js/script.js` – Scripts interactifs
- `photos/`, `videos/`, `thumbnails/` – Médias du site
- `attachments/` – Logo et ressources

## 🧩 Technologies utilisées
- TailwindCSS
- HTML5 & JavaScript Vanilla
- GitHub Pages (déploiement)
